JavaScript/TypeScript中的基本类型通常有如下几种：

  • 字符串
    • 常见单引号 - let string = 'single'
    • 类Java双引号 - let string = "double"
    • 模板式 - let string = `template`
    模板式中可通过${varName}来引用上下文的变量，以及可以在模板式中自然换行等价于\n。

    符合最新语法的转换方式：String(...)
  • 数值 - let number = 3.1415926
    要注意JS中的数值类型永远是64位浮点数。

    符合最新语法的转换方式：Number(...)
  • 布尔值 - let boolean = true
    符合最新语法的转换方式：Boolean(...)
  • 数组 - let array = [1,false]
    符合最新语法的转换方式：Array.from(arrayLike)
    符合最新语法的浅复制方式：[...array]
  • 对象 - let object = {a:1,b:false}
    符合最新语法的浅复制方式：{...object}
  • 函数 - let function = function(args){...}
    或箭头函数（引用上下文需要）：let arrFunction = (args)=>{...}
  • 唯一标识 - let symbol = Symbol('symbol')
    它等价于对它指针的唯一标识，除了获取它的指针，否则永远不可能有任何东西与它相等。
    Symbol('symbol') != Symbol('symbol')=>true

TypeScript的类型操作

在TypeScript中一个很重要的关键字便是type，通过它开发者还可以随心所欲的根据情况来控制Angular语言服务（非Angular项目的TS语言服务）的类型提示。

通常，在一个Angular应用程序中，type通常在以下场景中出现。

  • 变量需要明确声明类型，const example1: { a?: number; b?: string } = {};
  • 函数事先声明参数返回值类型，function example2(a?: number; b?: string) {...}
  • 使用泛型但不能自动类型断言时，array.reduce<{ a : number }[]>(...);
  • 手动进行类型断言的时候， filterFunctionA(test: unknown): test is number {...}
    array.filter(filterFunctionA);
