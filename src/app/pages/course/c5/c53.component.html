JavaScript/TypeScript中的基本类型

通常有如下几种：

  • 字符串
    • 常见单引号 - let string = 'single'
    • 类Java双引号 - let string = "double"
    • 模板式 - let string = `template`
    模板式中可通过${varName}来引用上下文的变量，以及可以在模板式中自然换行等价于\n。

    符合最新语法的转换方式：String(...)
  • 数值 - let number = 3.1415926
    要注意JS中的数值类型永远是64位浮点数。

    符合最新语法的转换方式：Number(...)
  • 布尔值 - let boolean = true
    符合最新语法的转换方式：Boolean(...)
  • 数组 - let array = [1,false]
    符合最新语法的转换方式：Array.from(arrayLike)
    符合最新语法的浅复制方式：[...array]
  • 对象 - let object = {a:1,b:false}
    符合最新语法的浅复制方式：{...object}
  • 函数 - let function = function(args){...}
    或箭头函数（引用上下文需要）：let arrFunction = (args)=>{...}
  • 唯一标识 - let symbol = Symbol('symbol')
    它等价于对它指针的唯一标识，除了获取它的指针，否则永远不可能有任何东西与它相等。
    Symbol('symbol') != Symbol('symbol')=>true
